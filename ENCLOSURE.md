# Enclosure Design

A basic enclosure can be 3D-printed or laser-cut from 3mm acrylic.

## Recommended Design
- Base plate: 120mm x 80mm with mounting holes for ESP32 dev board and sensor breakouts
- Sensor mounts: Each MQ sensor has 3mm mounting holes. Space them at least 20mm apart for airflow.
- Lid: Vented top with 5mm holes in a grid pattern above sensors
- Fan mount: Optional 30mm 5V fan for active air exchange

## Airflow Considerations
- MQ sensors need exposure to ambient air — do not seal the enclosure
- The heater element gets hot (~80°C). Ensure at least 10mm clearance above each sensor
- For food testing: a small door or slot to insert samples without opening the main chamber
